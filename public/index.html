Projects

Data Mining over English Songs
Oct, 2013 - Apr, 2014
Demo

In this project, we analysed 1000 English songs. If a sequence of tones from lyrics is frequently matched with a specific melody of song, we called this a pattern. For every song, we collected a set of all frequent patterns. Then we found a final set of all frequent patterns over all songs. Our program can analyse any English sentences as input, and generate a melody according to the final set of frequent patterns.
Teammates: Jiabin HU, Jing ZHAO, Bo WANG

iSearch
Oct, 2013 - Nov, 2013, Search Engine Course Project
Demo

In this project, we built a search engine. We indexed 1000 pages and rank the page by combining scores of vector space model and scores of Page Rank. Long phrases and word suggestion were also implemented.
Teammates: Min XIE, Jia TU

Performance analysis of Android application (a preliminary study)
Jun, 2013 - Jul, 2013, Undergraduate Research Opportunity Program
Detail (PDF)

Many Android applications nowadays still suffer from the slowness problem in varying degrees. By default, all components of the same application run in the single 'main' thread (called Single Thread Policy). Our porject is to develop a tool to identify possible methods that have potential threat of causing slowness.
Teammates: Yingjun Lv
